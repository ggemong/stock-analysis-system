## 파일 구조

```
stock-analysis-system/
├── README.md               # 프로젝트 개요
├── GUIDE.md               # 상세 사용 가이드
├── CHANGELOG.md           # 변경 로그
├── VALIDATION_REPORT.md  # 검증 리포트
├── PORTFOLIO.md           # 포트폴리오 구성 ⭐
├── TELEGRAM_EXAMPLE.md    # 텔레그램 메시지 예시 ⭐

├── config/
│   ├── stocks.json        # 13개 종목 등록됨
│   └── settings.json
├── src/
│   ├── collectors/
│   ├── analyzers/
│   ├── formatters/
│   └── notifiers/
└── main.py
```

## 주요 기능

✅ 미국 주식 분석 (AAPL, MSFT, NVDA, AMZN, META, TSLA)  
✅ 한국 주식 분석 (삼성전자, SK하이닉스, 현대차)  
✅ ETF 분석 (TIGER 리츠, ACE 금현물)  
✅ 암호화폐 분석 (BTC, ETH)  
✅ RSI + 과열/침체 상태  
✅ 정배열/역배열 감지  
✅ 골든크로스/데드크로스  
✅ 이격도 분석  
✅ 장기 이평선 정보 (MA20/50/200) ⭐  
✅ 텔레그램 자동 발송  
✅ GitHub Actions 자동화
