# 📊 VIX 지표 가이드

**업데이트:** 2026-01-30  
**버전:** 1.0

---

## 🎯 VIX란?

**VIX (Volatility Index)** = 변동성 지수 = 공포 지수

- 시카고 옵션거래소(CBOE)에서 산출
- S&P 500 지수 옵션의 내재 변동성 측정
- 향후 30일간 시장 변동성 예상치
- "시장의 공포 온도계"

---

## 📈 VIX 수준별 의미

### 😌 VIX < 15: 극도의 안도

**시장 심리:**
- 낙관적 분위기
- 변동성 극히 낮음
- 투자자 안심 상태

**투자 전략:**
- ✅ 주식 매수 우호적
- ✅ 레버리지/성장주 선호
- ⚠️ 과도한 낙관 주의
- ⚠️ 금, 방어주 매력 낮음

**예시 상황:**
- 경제 호황기
- 기업 실적 호조
- 정치적 안정

**주의사항:**
```
VIX가 너무 낮으면 오히려 위험!
→ 과도한 낙관 = 조정 가능성 ↑
→ "폭풍 전 고요" 일 수 있음
```

---

### 😐 VIX 15-25: 정상 범위

**시장 심리:**
- 건강한 시장
- 적당한 변동성
- 균형잡힌 투자심리

**투자 전략:**
- ✅ 정상적인 투자 환경
- ✅ 기본적 분석 중시
- ✅ 추세 추종 전략 유효
- ⚪ 특별한 헤지 불필요

**예시 상황:**
- 평소 시장 상황
- 예측 가능한 움직임
- 정책 안정기

**포트폴리오 전략:**
```
- 주식: 60-70%
- 채권/안전자산: 20-30%
- 대안자산: 10%
```

---

### 😰 VIX 25-35: 경계 및 공포

**시장 심리:**
- 불확실성 증가
- 변동성 급등
- 투자자 불안

**투자 전략:**
- ⚠️ 리스크 관리 강화
- ⚠️ 포지션 축소 검토
- ✅ 금 ETF 비중 확대
- ✅ 방어주/배당주 선호
- ❌ 레버리지 자제

**예시 상황:**
- 경제 지표 악화
- 지정학적 리스크
- 금리 급등
- 기업 실적 우려

**포트폴리오 조정:**
```
AS-IS (정상)        TO-BE (경계)
주식: 70%      →    주식: 50%
채권: 20%      →    채권: 25%
금: 5%         →    금: 15%
현금: 5%       →    현금: 10%
```

**구체적 행동:**
1. 손절 라인 재점검
2. 과열 종목 익절
3. 금 ETF 추가 매수
4. 변동성 큰 종목 축소

---

### 😱 VIX > 35: 패닉 (Panic)

**시장 심리:**
- 극도의 공포
- 패닉 셀링
- 시장 붕괴 우려

**투자 전략:**
- ❌ 추격 매수 금지
- ❌ 레버리지 절대 금지
- ✅ 현금 비중 최대화
- ✅ 금 ETF 최대 비중
- 🎯 매수 기회 준비 (역발상)

**예시 상황:**
- 금융위기 (2008년: VIX 80)
- 코로나 팬데믹 (2020년: VIX 82)
- 블랙먼데이
- 전쟁/테러

**포트폴리오 조정:**
```
극단적 방어 모드

주식: 30% (우량주만)
채권: 30%
금: 30%
현금: 10%

또는

역발상 기회 포착
→ 우량주 분할 매수 준비
→ 단, 손절 라인 반드시 설정
```

**주의사항:**
```
⚠️ VIX > 35는 극히 드문 상황
⚠️ 패닉에 동참하지 말 것
⚠️ 반등 시점을 미리 준비
✅ 역사적으로 최고의 매수 기회
```

## 📊 VIX와 자산별 관계

### 주식 (역상관)
```
VIX ↑ = 주식 ↓
VIX ↓ = 주식 ↑

상관계수: -0.7 ~ -0.8
```

### 금 (정상관)
```
VIX ↑ = 금 ↑
VIX ↓ = 금 ↓

VIX 25 이상 시 금 매력 급증
```

### REITs (역상관)
```
VIX ↑ = REITs ↓
VIX ↓ = REITs ↑

VIX 높으면 부동산도 약세
```

### 암호화폐 (높은 변동성)
```
VIX ↑ = 비트코인 불확실
VIX ↓ = 비트코인 ↑ (리스크 온)

VIX 30 이상 시 암호화폐 고위험
```

## 🎯 실전 투자 전략

### VIX 15 미만 (안도)
```
✅ DO
- 성장주 매수
- 레버리지 ETF 고려
- 적극적 포지션

❌ DON'T
- 과도한 낙관
- 전량 매수
- 헤지 소홀
```

### VIX 15-25 (정상)
```
✅ DO
- 균형잡힌 포트폴리오
- 추세 추종
- 정상 리스크 관리

⚪ NEUTRAL
- 특별한 조치 불필요
```

### VIX 25-35 (공포)
```
✅ DO
- 포지션 축소 (30%)
- 금 ETF 확대
- 손절 라인 점검
- 방어주 비중 확대

❌ DON'T
- 추가 매수
- 레버리지
```

### VIX 35 초과 (패닉)
```
✅ DO (방어)
- 현금 비중 30%+
- 금 최대 비중
- 손절 엄격히

✅ DO (공격, 신중히)
- 우량주 분할 매수
- 장기 투자 관점
- 손절 라인 필수

❌ NEVER
- 전량 매수
- 레버리지
- 단기 투기
```

## 📈 역사적 VIX 사례

### VIX 80 이상 (역사적 대폭락)
- **2008년 11월**: 금융위기 (VIX 80)
- **2020년 3월**: 코로나 팬데믹 (VIX 82)
- → 이후 모두 강력한 반등

### VIX 50-70 (심각한 위기)
- 블랙먼데이, 9.11 테러
- 유럽 재정위기
- → 회복까지 수개월~1년

### VIX 10 미만 (역사적 낙관)
- **2017년**: VIX 9.14 (역대 최저)
- **2018년 2월**: 급등으로 조정
- → 과도한 안도 후 조정 발생

## 💡 실전 팁

### 1. VIX 모니터링
```bash
매일 체크: VIX 현재값
주간 체크: VIX 추세 (상승/하락)
월간 체크: VIX 평균 vs 역사적 평균
```

### 2. 포트폴리오 조정 시점
```
VIX 20 → 25 돌파: 경계 시작
VIX 25 → 30 돌파: 포지션 축소
VIX 30 → 35 돌파: 방어 모드
VIX 35 → 40 돌파: 극단적 방어 또는 역발상
```

### 3. 금 ETF 비중
```
VIX < 20: 금 5-10%
VIX 20-30: 금 10-20%
VIX 30-40: 금 20-30%
VIX > 40: 금 30-40%
```

### 4. 절대 하지 말 것
```
❌ VIX 30+ 에서 레버리지
❌ VIX 급등 중 물타기
❌ VIX만 보고 판단 (다른 지표도 확인)
```

## 🔗 다른 지표와 함께 보기

### VIX + RSI
```
VIX 높음 + RSI 과매도 = 강력한 매수 신호
VIX 낮음 + RSI 과매수 = 조정 가능성
```

### VIX + 금리
```
VIX 높음 + 금리 하락 = 경기침체 우려
VIX 낮음 + 금리 상승 = 경기 과열
```

### VIX + 환율
```
VIX 높음 + 달러 강세 = 신흥국 위험
VIX 낮음 + 달러 약세 = 신흥국 기회
```
